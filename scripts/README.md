### Scripts

- `install.sh`: Installs all prerequisites for the study, such as `git`, `cmake`, `lm-sensors`, etc.
- `install_dependencies.sh`: Installs additional dependencies for the programming languages in this study to function properly. 
- `measure.sh`: Executes experiments on programs.
- `run_powercap_calibration.sh`: Runs powercap calibration.
- `generate_inputs.sh`: Generates needed iput files for certain benchmarks.
- `erlang_setup.sh`: Installs a specific version of Erlang with and without JIT.
- `java_setup.sh`: Install a specific version of Java.
- `javascript_setup.sh`: Install a specific version of Node.
- `lua_setup.sh`: Install a specific version of Lua and LuaJIT.
- `php_setup.sh`: Install a specific version of PHP.
- `python_setup.sh`: Install a specific version of Python3 and Pypy3.
- `ruby_setup.sh`: Install a specific version of Ruby with the JIT compilation enabled.
